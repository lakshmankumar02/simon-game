Game rule is so simple. In level 1, a button will flash then player(you) have to click that right button. If the clicked button is wrong then the game will be over but if it is right then the level changed into higher one. Now again, a button will flash but player(you) have also to click that previous button. e.g. If red button glows first time then you have to click red button and second time if red button again glows then you have to click red button twice and so on.
Remember the previous sequence.
